⚫️ THE ULTIMATUM GAME ( ဒါမှ မယူရင် ၊ ဘာမှ မယူနဲ့ )

သူငယ်ချင်း ၃ ယောက်
ကောင်းပေ့ လှပေ့ဆိုတဲ့ ဆိုတဲ့
ဆိုင်ကြီးမှာ ညစာ အတူတူ စားကြတယ်

ခဏလောက်နေ
ပိုက်ဆံရှင်းကြတော့
စားပွဲထိုးက ၉သောင်းကျတယ်ပြောတယ်

ဒါဆို တစ်ယောက်ကို ၃သောင်းပေါ့လို့ပြောတော့နောက်တစ်ယောက်က လိုက် သဘောတူတယ်
အဲ့အချိန်မှာ ကျန်တဲ့ သူငယ်ချင်းတစ်ယောက်က

⚫️ ငါက ဘာမှ မှ သိပ်မစားတာ အဲ့တော့
ငါ ၁သောင်းပဲ စိုက်နိုင်မယ်လို့ပြောတယ်

အဲ့ဒီလိုနဲ့ စကားများကြရော

ဒီဘက်ကလည်း မရဘူး
တစ်ယောက် သုံးသောင်း
ဟိုဘက်က လည်း တစ်သောင်းမှ တစ်သောင်း

ဒါမှ မယူရင် တပြားမှ မပေးနိုင်ဘူးလို့ပြော
အဲ့ဒီလိုနဲ့ အခြေအတင်ဖြစ်ကြရော

⚫️ပိုက်ဆံက အဓိက မဟုတ်ပါဘူး
အဓိကက မတရားဘူးလို့ ခံစားရတာ

တစ်ဖက်ကလည်း မစားဘဲနဲ့ အများကြီးပေးရင်
မတရာဘူးခံစားရတယ်
တစ်ဖက်ကလည်း သူများဝေစုကို
ခေါင်းခံ ပြီး ပိုမစိုက်ချင်ဘူး

တကယ်တော့
Rationally တွေးကြည့်ရင်

  • ၃သောင်းသမားက ပိုစိုက်ရတဲ့အခါ
    ၁သောင်းပဲ ပိုစိုက်ရမှာပါ
  • ဟို ၁ဦးတည်း သမားကလဲ
    လူအများနဲ့နေရင်
    လိုက်လျောညီထွေရှိအောင် နေသင့်တာ
    ဒါပေမဲ့ နှစ်ဖက်စလုံးက လုံးဝ မလျော့ဘူး

ပိုက်ဆံက အတွက် စကားများတယ်
ထင်ရပေမဲ့ တကယ်က မာနတွေကြောင့်
ကိုယ့် ယုံကြည်ချက်တွေကြောင့်
အဖွဲ့တွေ ကွဲကြတာ

⚫️ zero sum game နှင့် positive sum game

တဖွဲ့နိုင်ဖို့ တဖွဲ့က ရှုံးမှရတာကို
Zero sum game လို့ခေါ်တယ်

ဘောလုံးပွဲအတွက်
ဖလား လုကြသလိုပေါ့
တစ်ဖွဲ့က ချန်ပီယံဖြစ်ဖို့ အတွက်
တစ်ဖွဲ့က ရှုံးပေးရမယ်

ဒါပေမဲ့ zero sum လို့
ထင်ယောင်ထင်မှား ဖြစ်စေတာတွေလဲရှိတယ်
ဥပမာ ကျောင်းတုန်းက
အဆင့်ပြိုင်ကြတယ်ပေါ့

⚫️တစ်ယောက် အဆင့် ၁ ရဖို့အတွက်
ကျန်တဲ့သူတွေက အဆင့် ၁ မရမှ ဖြစ်မယ်

ဒါပေမဲ့ G12 ရောက်တဲ့အခါ
ပြိုင်နေတာထက်ပူးပေါင်းတာက
ပို အကျိုးရှိတယ်

ဘာလို့လဲဆိုတော့
ပူးပေါင်းလို့ သူငယ်ချင်းနှစ်ယောက်
အပြန်အလှန် အမှားတွေ ထောက်ပြရင်း
နဂိုတစ်ယောက်တည်း လုပ်နေတာထက်တော့
ပိုတိုးတက်သွားရင်
ဒါဟာ Win-Win ပဲ
ဒါမျိုးကို positive sum game လို့ခေါ်တယ်

တကယ်တော့ တချို့ game တွေက
မပြိုင်ဘဲ ပူးပေါင်းသင့်တယ်
