Linear Career Path

မူကြိုတက်မယ် ပြီးရင် မူလတန်း အလယ်တန်း
အဲ့ဒီကနေ အထက်တန်း
ပြီးတော့ Universityကနေ ဘွဲ့ရပြီးတော့
ကိုယ်လိုချင်တဲ့ " လိုင်း (Major) " ကို ရွေးမယ်

အဲ့ဒီလိုင်းနဲ့ တစ်သက်လုံးလုပ်ရင်းနဲ့
ပိုက်ဆံတွေအများကြီးရှိလာတဲ့အချိန်ကြ
အနားယူကြမယ်
ဒါကို Linear Career Path လို့ ခေါ်တယ်

တကယ်တော့ ဘဝက Linear မဟုတ်ဘူး

၈၀% သော ကျောင်းသား ကျောင်းသူတွေက
သူတို့ ဘဝဟာ မျဉ်းဖြောင့်အတိုင်းပဲ
တရွေ့ရွေ့ လေးသွားနေမယ်လို့ ပုံသေတွက်ထားကြတယ်

ဒါပေမယ့် ဘဝက Linear မဟုတ်ဘူး
ဒါကို သိစေမြင်စေချင်တယ် ...

。 。 。

အခုစာဖတ်နေတဲ့သူရဲ့
ဘဝက လိုတရနေတယ်
အခက်အခဲ ဆူးငြောင့်ကသင်းမရှိဘဲ
တရွေ့‌တရွေ့နဲ့ ဖြတ်သန်းနေရတယ်ဆိုရင်

ဂုဏ်ယူပါတယ်
သင်က အင်မတန်ကံကောင်းတဲ့ 0.0001% သော
လူတန်းစားမှာပါတယ်

ဒါမှမဟုတ် အသက်အရမ်းငယ်လွန်းလို့
ဘဝကို သေချာ မဖြတ်သန်းရသေးလို့ဖြစ်ပါလိမ့်မယ်

。 。 。

အဲ့သလိုမှမဟုတ်ဘဲ
ဘဝရဲ့ လွဲချော်မှုတွေနဲ့ ရင်ဆိုင်ခဲ့ရတယ်
ရင်ဆိုင်နေရတယ်ဆိုရင်တော့
ကျနော်ပြောတာ သဘောပေါက်မှာပေါ့

  • ရှုံးနိမ့်ခြင်းတွေရှိမယ်
  • လွဲချော်ခြင်းတွေရှိမယ်
  • မပိုင်ဆိုင်ရခြင်းတွေရှိမယ်
  • မဖြစ်ပါစေနဲ့ ရေရွတ်ခြင်းတွေ တည့်တိုးဖြစ်လာတာရှိမယ်
  • ဖြစ်ချင်တာတွေ မဖြစ်ရတာလဲရှိမယ်
  • ဘဝလမ်းကြောင်းကြီး ချော်ထွက်သွားခဲ့တာရှိမယ်

သေးသေးလေးမဟုတ်ဘဲ
အဖြစ်အပျက် Event အကြီးကြီးတွေပေါ့

。 。 。 。
ဘဝမှာ သင်မမြင်ရတဲ့အချိုးအကွေ့တွေ အများကြီးရှိနိုင်ပါတယ်

လမ်းဆုံ လမ်းခွဲတွေ

ဒီအဖြစ်အပျက်ကနေ
လမ်းဆုံ လမ်းခွဲတွေ ပေါ်ပေါက်လာတယ်
ဆိုကြပါစို့

ဒီ ကလေးအနေနဲ့ပြောရမယ်ဆိုရင်
ဆရာဝန်ဖြစ်သွားခြင်းက လမ်းခွဲတစ်ခု
ဆရာဝန်မဖြစ်ခြင်း လမ်းခွဲတစ်ခု

ဆရာဝန်ဖြစ်သွားတဲ့လမ်းကို
သူစိတ်ကူးယဉ်ပုံဖော်လို့ရပေမယ့်

ဆရာဝန်မဟုတ်တဲ့လမ်းကို သူ ပုံမဖော်နိုင်ဘူး
ဒါပေမယ့် တကယ့်လက်တွေ့မှာ
ဆရာဝန်မဖြစ်ခြင်းက
ပိုကောင်းတဲ့လမ်းဖြစ်နေမယ်ဆိုတာ

。 。 。

ဘယ်သူမှ မပြောနိုင်ဘူး
ဆရာဝန်အမှတ် မမီလို့ ကျောင်းဆရာလုပ်တယ်
ကလေးတွေနဲ့ စာသင်ကြ ပျော်ကြပါးကြနဲ့

ဘဝရဲ့ အဓိပ္ပာယ်အသစ်တစ်ခုကို ရှာတွေ့ပြီး
ပို လေးနက်တဲ့ အရာကို ပိုင်ဆိုင်ရမယ်
ဒီလိုမျိုးပေါ့

ကျနော်လည်း စာသင်မယ်လို့ရည်ရွယ်ချက်မရှိ၊
ခံယူချက်မရှိဘဲနဲ့ အနီးနားကလေးတွေကို
စာသင်ဖြစ်ရာက စာသင်ရတာကိုချစ်သွားတာပဲ
ဒါကြောင့် ဘဝကို ဘာကြီးဖြစ်ရမယ်
Identity တစ်ခုထဲမှာပဲ ပိတ်မထားသင့်ဘူး

。 。 。

ကျနော့်အနေနဲ့ ပြောပြရမယ်ဆိုရင်

စာရေးတယ်
Utube Video လုပ်တယ်
စာသင်တယ်
Code ရေးတယ်

စာအုပ်တွေထဲမှာ နစ်မြောမယ်

ကိုယ်ဟာ ဘာကြီးပါဆိုတဲ့
identity ခံယူချက် ဘောင်တစ်ခုတည်းမှာ
ကိုယ့်ကိုယ်ကို ပိတ်မထားရင်
မတူညီတဲ့ အရာတွေ ပေါင်းစပ်ပြီး
မတူညီတဲ့ Life Style တွေကို ဖြတ်သန်းလို့ရတယ်
ဘဝ ပေါင်းများစွာမှာ စီးမြောရှင်သန်လို့ရတယ်
အဲ့ဒီလိုဆို တဖြည်းဖြည်းနဲ့ ကိုယ့်ရဲ့ တန်ဖိုးက တက်လာမယ်

。 。 。

ဖြစ်ချင်တာ မဖြစ်မလာရင်
စိတ်ညစ်စရာမရှိဘူးရယ်
ပိုကောင်းတဲ့ အခွင့်အရေးတွေလာဖို့ရှိလို့
အခုဟာကို လက်လွတ်ဖို့ဖြစ်လာတာ
လို့ပဲတွေးလိုက်

ဘာလို့အဲ့ဒီလိုပြောရတာလဲဆိုတော့
ဘဝက Linear မဟုတ်ဘဲ
ပုံမှာ ပြထားတဲ့ သစ်ပင်နဲ့တူလို့

ပင်စည်တစ်ခုတည်းကနေ အစပြုပေမယ့်
ဘဝ Moment တွေကနေ ကိုင်းဖြာသွားတယ်
လွဲချော်ခြင်းတွေ ရွေးချယ်ခြင်းတွေ လုပ်ရတိုင်းလုပ်ရတိုင်း

ကိုင်းဖြာသွားရင်း ကိုင်းဖြာသွားရင်း
ဘယ်လောက်ထိ ဖြာပြီး
ဘာတွေဖြစ်မယ်ဆိုတာ ဘယ်သူမှ မပြောနိုင်ဘူး

‌ဒါကြောင့် ဒီပုံထဲက သစ်ပင်လေး အရမ်းလှနေတာပေါ့
အကိုင်းမရှိဘဲ ဖြောင့်ဖြောင့်တန်းတန်းကြီးသာဆို
